# Quasar App (quasar-vue-web)

Este proyecto fue realizado con Quasar framework, vue 3, pinia

## Install the dependencies
```bash
npm install
```

### Start app
```bash
npm run dev
```

### Dependencies

This project use:
* Node 20.10.0
* Vue ^3.4.15
